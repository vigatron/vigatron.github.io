HEINBK NB986 386SX Encyclopedia Notebook is a rare,
obscure early-1990s Taiwanese notebook computer (circa 1990–1991)
manufactured by **Honotron Corporation** (Taipei, Taiwan).

Its FCC ID is **HEINBK** (Grantee Code HEI, Product Code NBK; approved October 17, 1991),
and the model appears to be a variant of the **NB986** / **NBK** series,
marketed under the “Encyclopedia Notebook” name
(sometimes styled as Encyclopedia 386SX or Encyclopedia x386).
It was also rebadged/sold identically as the Cybermate 386SX in some markets.
